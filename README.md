## 🌟 Features / Características

### 🇬🇧 English
- **Interactive Installer**: Auto-detects OS and sets up custom paths/commands.
- **Ollama AI Integration**: Local LLM chat processing with emotion detection.
- **Glassmorphism UI**: Modern, translucent interface with soft corners (24px).
- **Smooth Chaining**: Support for up to 5 sequential speech bubbles.
- **Internal Thoughts**: Analytical process visualization in cyan.
- **Performance Modes**: Low, Medium, and High settings.

### 🇪🇸 Español
- **Instalador Interactivo**: Auto-detecta el SO y configura rutas y comandos.
- **Integración con Ollama**: Procesamiento de chat local con detección de emociones.
- **Interfaz Glassmorphism**: Diseño moderno y translúcido con bordes redondeados (24px).
- **Encadenamiento Fluido**: Soporte para hasta 5 bocadillos de texto secuenciales.
- **Pensamientos Internos**: Visualización del razonamiento de la IA en color cian.
- **Modos de Rendimiento**: Ajustes Low, Medium y High para optimizar recursos.

## 🛠 Installation / Instalación

Choose your preferred method / *Elige tu método preferido*:

### 1. Interactive One-click Setup (Recommended)
This script will ask for your preferred path and command name.
**Linux/Mac:**
```bash
bash install.sh
```
**Windows/Any:**
```bash
python interactive_install.py
```

### 2. Manual Setup
```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
```

## 🚀 Execution / Ejecución

Start the avatar using your custom command (default is `makima`) or:
- **Bash Alias:** `makima`
